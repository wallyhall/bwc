## AI free code

This code is hand-written using traditional 2000's resources like tldp.org and manual pages.

Any hallucinations are human.

## Portability

As of March 2026, nothing has been extensively tested on anything other than Ubuntu-WSL2.  The intention is for this to be fairly portable wherever Bash is found on a POSIX(-ish) compliant platform (contemporary BSDs, Linux, etc).

## The spec is the functionality

How it works is how it works.  The UX is opinionated and built around the author - who doesn't benefit from many accessibility functionalities.  He has plans to include color-reduced/high-contrast modes and the alike in due course.
